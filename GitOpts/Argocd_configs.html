Reconciliation loop  

  • Reconciliation loop is how often does argocd sync with the git server > by default every 3 min, and can be configured via argocd-cmd-params-cm configmap
kubectl -n argocd patch configmap argocd-cmd-params-cm --patch='{"data":{"server.repo.server.timeout.seconds":"300s"}}'
kubectl -n argocd rollout restart deploy argocd-repo-server
