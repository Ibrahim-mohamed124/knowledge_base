Reconciliation loop  

  • Reconciliation loop is how often does argocd sync with the git server > by default every 1 min, and can be configured via argocd-cmd-params-cm configmap
  • server.insecure: “true” to make the argocd server run in insecure mode
kubectl -n argocd patch configmap argocd-cmd-params-cm --patch='{"data":{"server.repo.server.timeout.seconds":"300s"}}'
kubectl -n argocd rollout restart deploy argocd-repo-server
