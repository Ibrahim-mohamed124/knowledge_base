Reconciliation loop and Webbooks configurations 

  • Reconciliation loop is how often does argocd sync with the git server > by default every 3 min, and can be configured via argocd-cm configmap timeout.reconciliation":"300s" parameter 
kubectl -n argocd patch configmap argocd-cm --patch='{"data":{"timeout.reconciliation":"300s"}}'
kubectl -n argocd rollout restart deploy argocd-repo-server
