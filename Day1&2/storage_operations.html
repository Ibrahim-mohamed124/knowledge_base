Configuring the Internal Image Registry to Use OpenShift Data Foundation

  • The image registry is integrated into the cluster authentication and authorization systems.
  • OpenShift stores the container image data in the following locations:
    • The image data in a configurable storage location
    • The image metadata as standard API resources, such as images and image streams
