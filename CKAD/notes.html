  • CMD vs ENTRYPOINT
    • CMD is replaced if a new  command is entered in the run
    • ENTRYPOINT, new  commands will be appended  only  in the run
      • to change the entrypoint use --entrypoint in docker
    • To configure a default argument to the entrypoint use CMD, and it will be changed if  a new command is entered in the run  like  args in the k8s (override the CMD)
    • command in k8s overrides the entrypoint
