• CIS benchmarks:
    • security benchmark: the baseline practices to secure an environment
    • CIS: center for Internet security
  • Kubernetes CIS benchmarks:   Best practices for the cluster
  • kube-bench: automation tool to automatically scan k8s clusters against CIS benchmarks
kube-bench
  • Download the binary
curl -L https://github.com/aquasecurity/kube-bench/releases/download/v0.12.0/kube-bench_0.12.0_linux_amd64.deb -o kube-bench_0.12.0_linux_amd64.deb

 For ubuntu download it using the dpkg command

sudo dpkg -i kube-bench_0.12.0_linux_amd64.deb

 Run a test

kube-bench run
