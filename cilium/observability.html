Hubble

  • enable real-time traffic inspection across the cluster
    • analyze and track all the flows
    • operate at the node, cluster, across clusters levels
  • Hubble server runs on every node that is integrated in the cilium agent pod
  • the Hubble relay only one server across the cluster
  • Hubble CLI communicate with the hubble relay
  • the Hubble UI pod introduce a GUI
  • To enable Hubble > enabled by default
cilium hubble port-forward
huble observe --help

Prometheus & Garafana

  • To expose metrics:
    • prometheus.enabled: true
    • operator.prometheus.enabled: true
    • hubble.metrics.enalbed: [array of metrics] and enalbedOpenMetrics: true
